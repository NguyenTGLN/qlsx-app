# Trang Cải Tiến — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Tab "Cải tiến" trong phân hệ Công việc: nhân viên gửi cải tiến kèm ảnh/video, app tự quy đổi giá trị thành tiền, quản lý chấm 5 tiêu chí → xếp loại A/B/C, bảng xếp hạng.

**Architecture:** Engine tính giá trị = hàm thuần trong `src/lib/caiTienValue.js` (TDD, không side effect). UI = 1 file tab `src/pages/tasks/CaiTienTab.jsx` theo pattern ChamCongTab (props `{me, users, perm}`, tự fetch qua `fetchAllRows`, tải mềm khi bảng chưa tạo). Schema = `sql/create_cai_tien.sql` idempotent + RLS theo claim `nv_id`/`nv_role` (pattern create_chuyen_can_ngoai_le.sql). Ảnh/video dùng lại bucket `task-attachments` + `AttachmentInput`.

**Tech Stack:** React (JSX), Supabase (anon + custom JWT), vitest, lucide-react.

**Spec:** `docs/superpowers/specs/2026-07-24-cai-tien-design.md` (đã duyệt). Mockup: `docs/mockups/cai-tien-ui-mockup.html`.
